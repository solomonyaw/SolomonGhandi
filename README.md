# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/066604a4-7bad-4f91-9efc-a9ef07758f0f

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

   # 🎧 Solomon Ghandi – AI Hip-Hop Project

**Solomon Ghandi** is an AI-powered hip-hop artist founded by **Solomon Yaw Adeklo**, a Ghanaian software developer passionate about merging artificial intelligence, philosophy, and music to create conscious and transformative hip-hop.

## 👤 About the Artist – Solomon Ghandi

**Solomon Ghandi** represents a new form of artistry where **AI meets consciousness**.  
He is not just a music persona, but a **symbol of intelligent, philosophical hip-hop** designed to challenge systems, provoke thought, and inspire change.

The name *Solomon Ghandi* originated as a nickname during secondary school in Ghana and has evolved into a powerful identity representing:
- Wisdom and knowledge  
- Peaceful but strategic resistance  
- Global consciousness through music  

As an AI-driven artist, Solomon Ghandi focuses on:
- Conscious and philosophical lyricism  
- Systems thinking and global awareness  
- Using music as a tool for education and transformation
